# Plan Wdrożenia: Aktualizacja Strony dla Klinik Stomatologicznych

## Cel
Dostosowanie strony internetowej pod kątem nowej niszy (B2B Stomatologia), poprzez zmianę komunikacji na język korzyści dla właścicieli klinik, przebudowę sekcji Hero, wyeksponowanie Kalkulatora ROI oraz optymalizację formularza kontaktowego.

## Lista Kontrolna (Checklista)

- [x] **1. Sekcja Hero (Główna)**
  - [x] Zmiana nagłówka (H1) na: "Pełny grafik. Zero pustych foteli."
  - [x] Zmiana podtytułu (subline) na: "Koniec z odwołanymi wizytami i chaosem w recepcji. Wdrożymy inteligentną automatyzację i AI skrojoną pod Twoją klinikę stomatologiczną, która zapełni grafik 24/7."

- [x] **2. Kalkulator ROI**
  - [x] Przeniesienie sekcji z kalkulatorem wyżej w strukturze strony (aby była bardziej widoczna). (Uwaga: jest już zaraz pod Hero).
  - [x] Zwiększenie interaktywności i intuicyjności interfejsu dla właścicieli klinik.
  - [x] Dodanie podsumowania przeliczanego na żywo w PLN (oszczędzony czas recepcji, odzyskane dochody z "no-shows").

- [x] **3. Język korzyści (B2B Stomatologia)**
  - [x] Przegląd i aktualizacja ogólnych opisów na stronie.
  - [x] Zmiana opisów na specyficzne dla gabinetów dentystycznych (np. integracja z kalendarzem, automatyczne przypomnienia SMS z opcją szybkiego odwołania wizyty, samoobsługowa rezerwacja dla pacjentów 24/7).

- [x] **4. Formularz Kontaktowy**
  - [x] Zmiana nagłówka nad formularzem na: "Zamów bezpłatną 15-minutową symulację grafiku dla Twojego gabinetu".
  - [x] Zmiana tekstu przycisku (CTA) na biznesowy, np.: "Zamów bezpłatną konsultację" (unikanie słowa "wizyta").
